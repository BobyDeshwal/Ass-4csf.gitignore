Project Title
Simple Arithmetic Calculator using Python

Project Description
This mini-project is a Python-based arithmetic calculator that performs basic mathematical operations such as addition, subtraction, multiplication, and division. The program takes input from the user, processes the selected operation, and displays the result.
It is designed to help beginners understand core programming concepts including:
Variables
Input/output handling
Conditional statements (if-elif-else)
Functions (optional)
Error handling

This project is fully compatible with VS Code, allowing students to write, run, and test their code efficiently using the integrated terminal.

Installation Instructions (VS Code Setup)

1️⃣ Install Python
Download & install Python from the official website:
https://www.python.org/downloads
(Ensure “Add to PATH” is checked during installation.)

2️⃣ Install Visual Studio Code (VS Code)
Download & install VS Code:
https://code.visualstudio.com/

3️⃣ Install Python Extension in VS Code
Inside VS Code:
1. Go to Extensions Panel (left sidebar)
2. Search “Python”
3. Click Install

4️⃣ Create Your Project Folder
Create a folder named:
CalculatorProject

Inside it, create a file named:
csf python calculator.py

Usage Instructions

1️⃣ Open the Project in VS Code
Click File → Open Folder
Select CalculatorProject
Open csf python calculator.py

2️⃣ Run the Program
In VS Code:
Open Terminal → New Terminal
Run:
python csf python calculator.py

3️⃣ Provide Input
The program will:
Ask you to enter two numbers
Ask you to choose an operation (+, -, *, /)
Display the result


Example:
Enter first number: 10
Enter second number: 5
Choose operation (+,-,*,/): *
Output: 50
